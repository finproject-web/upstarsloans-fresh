Deploy UPSTART LOANS Website

Option 1: GitHub Pages (Free & Easy)

  1. Go to github.com and create account
  2. Create new repository named "upstartloans"
  3. Upload your static.html and images.png files
  4. Go to Settings > Pages in your repo
  5. Select "Deploy from branch" and main branch
  6. Your site will be live at: yourusername.github.io/upstartloans

Option 2: Netlify (Free & Fast)

  1. Go to netlify.com
  2. Drag and drop your static.html file
  3. Get instant URL like random-name.netlify.app
