# UPSTART LOANS Website

## Quick Deployment Instructions

### Option 1: Netlify (Easiest - 2 minutes)
1. Go to https://netlify.com
2. Drag and drop `static.html` file
3. Get instant live URL

### Option 2: GitHub Pages (Free)
1. Create GitHub account
2. Create new repository
3. Upload `static.html` and `images.png`
4. Enable GitHub Pages in settings

### Option 3: Vercel (Current)
- Your current site is at upstarsloans.vercel.app
- To update: Login to Vercel dashboard and upload new files
